# DevUtils Pro - 用户使用手册 (User Manual)

## 简介 (Introduction)
DevUtils Pro 是一款专为开发者设计的全能工具箱，提供格式化、转换、生成、比较等多种实用功能。本手册将帮助您快速了解并使用各个工具。

---

## 🛠️ 功能列表 (Features)

### 1. 格式化工具 (Formatters)
*   **JSON 格式化 (JSON Formatter)**
    *   **功能**: 输入压缩或混乱的 JSON 字符串，输出美化后的格式。支持自动修复简单的转义字符（如日志中的 JSON）。
    *   **使用**: 将 JSON 粘贴到左侧框，右侧即时显示结果。如果 JSON 有误，会显示 "Invalid JSON"。
*   **XML 格式化 (XML Formatter)**
    *   **功能**: 将单行 XML 转换为缩进良好的多行 XML。
    *   **使用**: 输入 XML，点击 "格式化 XML" (Format XML) 按钮。

### 2. 转换工具 (Converters)
*   **Base64 转换 (Base64 Converter)**
    *   **功能**: 支持文本和文件的 Base64 编码与解码。
    *   **使用**: 
        *   **文本**: 选择 "编码" 或 "解码" 模式，在输入框输入文本。
        *   **文件**: 在 "编码" 模式下点击 "上传文件" 按钮，文件将自动转换为 Base64 字符串。
*   **大小写转换 (Case Converter)**
    *   **功能**: 将文本转换为大写、小写或首字母大写。
    *   **使用**: 输入文本，点击下方对应的按钮。
*   **文本转 ASCII/Unicode**
    *   **功能**: 查看字符对应的二进制 ASCII 码或 Unicode 转义序列 (`\uXXXX`)。

### 3. 生成器 (Generators)
*   **ASCII 艺术生成 (ASCII Art Gen)** (需 API Key)
    *   **功能**: 利用 AI 将普通文本转换为酷炫的 ASCII 字符画。
    *   **注意**: 这是一个在线功能，需要配置 Google Gemini API Key 才能使用。
*   **文字图标生成 (Text Icon Gen)**
    *   **功能**: 快速制作简单的文字头像或图标（如 "JS"）。
    *   **使用**: 输入文字（建议 1-3 个字符），选择文字颜色和背景颜色，点击 "下载" 保存为 PNG。
*   **Base64 图片 (Base64 Image)**
    *   **功能**: 图片转 Base64 字符串，或 Base64 字符串还原为预览图。
    *   **使用**: 拖拽图片到上传区，或将 Base64 字符串粘贴到文本框。
*   **JSON 转 Java (JSON to Java)**
    *   **功能**: 将 JSON 对象自动转换为 Java Bean (POJO) 实体类代码。
    *   **使用**: 输入 JSON，设置类名，右侧自动生成包含 Getter/Setter 的 Java 代码。

### 4. 比较工具 (Comparators)
*   **JSON/文本 比较 (Diff)**
    *   **功能**: 对比两段文本或 JSON 的差异。
    *   **使用**: 分别在左右输入框粘贴内容，下方红色代表删除，绿色代表新增。

## ⚙️ 常见问题
**Q: 为什么 ASCII 生成无法使用？**
A: 该功能依赖 AI 模型，如果您在本地部署或自行编译，请确保在 `.env` 文件中配置了有效的 `VITE_API_KEY`。

**Q: 插件窗口太小怎么办？**
A: 如果是以 Chrome 插件方式运行，窗口大小是固定的。如果需要在更大屏幕上使用，建议部署为 Web 应用版本。
